# ConecTEA - Site Institucional

Site institucional moderno e acessível desenvolvido em React.js, JavaScript e Tailwind CSS para o projeto ConecTEA em parceria com a Associação Espaço do Autista (Fortaleza/CE).

## 📋 Sobre o Projeto

O ConecTEA é um projeto acadêmico voltado a fortalecer o atendimento a pessoas autistas através de tecnologia e apoio institucional. O site apresenta o projeto, dá visibilidade à parceria com o Espaço do Autista, facilita doações e capta voluntários.

## 🎨 Características

- ✅ Design moderno e responsivo (mobile-first)
- ✅ Acessibilidade (contraste adequado, navegação por teclado, textos alternativos)
- ✅ Animações discretas e microinterações
- ✅ 7 páginas completas com conteúdo estruturado
- ✅ Sistema de navegação fluida
- ✅ Barra de progresso de scroll
- ✅ Cards com efeitos hover
- ✅ FAQ com accordion e busca
- ✅ Formulário de contato
- ✅ CTAs estratégicos em toda jornada

## 📄 Páginas

1. **Home** - Apresentação do projeto, parceria e CTAs principais
2. **ConecTEA** - Sobre o projeto acadêmico, objetivos e transparência
3. **Espaço do Autista** - História, missão, público e desafios da Associação
4. **Doações** - Como doar (PIX, transferência, materiais)
5. **Voluntariado** - Áreas prioritárias, perfis e cadastro
6. **FAQ** - Perguntas frequentes com busca e categorias
7. **Contato** - Formulário e canais de comunicação

## 🚀 Como executar

```bash
# Instalar dependências
npm install

# Executar em desenvolvimento
npm run dev

# Build para produção
npm run build

# Preview do build
npm run preview
```

## 📝 Placeholders a Preencher

Antes de colocar o site no ar, preencha os seguintes dados oficiais:

### Logos
- Logo do ConecTEA
- Logo do Espaço do Autista

### Contatos
- E-mail do ConecTEA
- E-mail da Associação
- Telefone da Associação
- Endereço físico

### Doações
- Chave PIX oficial
- Dados bancários (banco, agência, conta, CNPJ)
- Outras formas de doação
- Instruções para doação recorrente
- Lista de materiais aceitos
- Link para relatórios de transparência

### Voluntariado
- Link do Google Forms para cadastro de voluntários

## 🎨 Cores do Projeto

### Primária (Azul)
- Usada em elementos principais, botões e destaques
- Representa confiança e profissionalismo

### Accent (Laranja)
- Usada em CTAs secundários e elementos de destaque
- Representa energia e ação

## 📱 Responsividade

O site foi desenvolvido com abordagem mobile-first e está otimizado para:
- Smartphones (320px+)
- Tablets (768px+)
- Desktops (1024px+)
- Telas grandes (1280px+)
